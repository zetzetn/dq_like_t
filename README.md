# ドラクエ風アプリ
ドラクエの戦闘画面を模写したWEBアプリ

# Dependency
ruby 2.5.1
rails 5.2.4
mysql2
bootstrap
github-flow

# Setup
ヘロクにデプロイしたWEBアプリとなります。
セットアップは必要ありません。

# Usage
デモでも戦闘に参加することが可能です。
戦いの記録をつけることが可能です。
